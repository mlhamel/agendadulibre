{% trans "Exportation des données" %}

{% blocktrans %} Il y a plusieurs manières d'accéder à l'information de l'Agenda du libre du Québec. L'une d'elle est la page Web, les autres sont les fils RSS et les calendriers iCal. Les fils RSS sont utilisables dans un navigateur Web ou une application de lecture de nouvelles. Les calendriers iCal sont utiles dans une application de calendrier: les événements s'y ajouteront automatiquement à partir du site Web.

Fils RSS

Chaque fils RSS liste les 10 événements pour les prochains jours ou les 10 plus récents événements ajoutés, pour tous le Québec ou pour une région donnée. En vous abonnant à un fils régional, vous recevrez des informations sur les événements de votre région à portée locale, mais également les événements à portée nationale. {% endblocktrans %}
